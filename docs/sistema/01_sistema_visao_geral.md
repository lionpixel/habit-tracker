# Sistema de Organização Pessoal — Visão Geral

## Objetivo
Conectar todos os módulos de vida em uma única engrenagem: do sonho ao resultado, passando por metas, tarefas, hábitos, finanças e saúde.

## Como se conecta
Este documento é o mapa-mestre. Cada módulo filho referencia este arquivo como contexto raiz.

## A Engrenagem

```
Sonhos (Quadro Visual)
  └── Metas Anuais
        └── Metas Trimestrais
              └── Metas Mensais
                    └── Metas Semanais
                          └── Tarefas Diárias
                                └── Hábitos Recorrentes
```

Módulos de suporte (alimentam os insights):
- **Controle Físico** → impacta metas de saúde
- **Controle Financeiro** → impacta metas de renda
- **Projetos** → alimenta tarefas semanais
- **Revisões** → fecha o loop e recalibra o sistema

## Princípios de Design

1. **Tudo conectado** — nenhum dado é ilhado
2. **Matemática automática** — o sistema calcula por semana, por dia
3. **Progresso visível** — % sempre exposto, status sempre claro
4. **Revisão fecha o loop** — semanal e mensal obrigatórios
5. **Escalável** — pronto para virar SaaS com Supabase
